总结

本文的想法较为简单，是将字符串的编辑距离推广至token列表的编辑距离，可以作为一种衡量字符串相似度的指标。

其它想法如下：

  1. 不仅字符串，token可以作为字符串相似度的衡量标准，在文本纠错中，拼音也可作为标准
  2. edit distance本质上是两个列表的编辑距离
  3. 利用预训练模型进行文本纠错，实质上是对token进行纠错
  4. 在中英文混合场景也许有用
